# devtrails-project 
# AI-Based Parametric Insurance Platform

## Problem
Delivery workers fake GPS use karke fraud claims kar rahe hain, jisse platform ka loss ho raha hai.

## Solution
Ek intelligent system jo real vs fake claims detect kare using multi-layer verification and AI-based fraud detection.

---

## Adversarial Defense & Anti-Spoofing Strategy

 1. Differentiation
- GPS + movement pattern analysis
- Speed validation
- Sensor-based verification
- AI-based anomaly detection

 2. Data Used
- Device ID & IP tracking
- Weather API validation
- Route history
- Time clustering
- Multiple account detection

3. UX Handling
- Direct rejection nahi
- Flagged claims → verification
- Delayed payout
- Manual review

 4. Security
- Risk scoring system
- Fraud cluster detection
- Zero-trust architecture
